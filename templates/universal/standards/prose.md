# Prose Standard

Apply when your output is written content. Evaluate every piece against these criteria.

**Single Responsibility** — Each section has one job. If you need the word "and" to describe what a section covers, split it.

**DRY** — State each idea once. If the same point appears in two sections, consolidate. Restating a concept in different words is not emphasis — it is waste.

**Clarity** — Active voice. Present tense. Sentences under 25 words. One idea per paragraph. Define technical terms on first use.

**Source of Truth** — Every factual claim links to its source. No vague attributions ("experts say", "industry reports suggest"). Name the source or drop the claim.

**Progressive Disclosure** — Start simple, build to complex. Quick wins before deep dives. Don't front-load complexity that the reader hasn't been prepared for.

**Authentic Voice** — Write like a person with experience, not a model producing text. Vary sentence rhythm. Have a perspective. Acknowledge uncertainty when you have it. Catch and rewrite AI-pattern tells (detection rules live in the specialist, not here).
